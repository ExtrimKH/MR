# Витрина товаров

Простой сайт-каталог с закрытой админкой. Ты добавляешь товары (фото + цена + описание)
и редактируешь тексты «Как это работает» и «Куда отправлять машину» — посетители видят витрину.

## Как запустить

1. Открыть терминал (PowerShell) в этой папке.
2. Один раз установить зависимости:
   ```
   npm install
   ```
3. Запустить сайт:
   ```
   npm start
   ```
4. Открыть в браузере:
   - Витрина:  http://localhost:3000
   - Админка:  http://localhost:3000/admin

## Пароль администратора

По умолчанию пароль `admin123`. **Обязательно поменяй его.**
Запуск с своим паролем:

```
$env:ADMIN_PASSWORD="мой-секретный-пароль"; npm start
```

## Где что хранится

- `data/db.json` — все товары и тексты (это твоя «база»).
- `uploads/` — загруженные фотографии.

Чтобы сделать резервную копию — достаточно скопировать папки `data` и `uploads`.

## Что дальше (когда захочешь выложить в интернет)

Сейчас сайт работает только на твоём компьютере. Чтобы он был доступен всем по ссылке,
его нужно разместить на хостинге (VPS или облако). Скажи — помогу это настроить.
